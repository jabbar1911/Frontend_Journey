# Frontend Journey

A day-wise frontend development learning repository tracking my progress from HTML basics to building a real-world UI clone using only HTML5 and CSS3.

## Overview

This repository documents my journey learning frontend web development. Starting from basic HTML elements and progressing through CSS styling, layouts, and media handling, I've challenged myself to gradually build more complex projects. The repository showcases hands-on practice from Days 61 to 75, with each day focusing on specific concepts and techniques.

## Learning Journey Summary

I started with fundamental HTML concepts like structure, semantic tags, and basic elements. As I progressed, I incorporated CSS for styling and layout, learned to handle images, audio, and video elements, and eventually combined all these skills to build a real-world project—a Paytm Homepage Clone.

This learning approach helped me understand how web development skills build on each other, from writing clean HTML to creating visually appealing layouts with CSS.

## How to Run the Projects

### Option 1: View Individual Days
1. Navigate to the `frontend_practice/` folder
2. Open any `Day_XX` folder
3. Open the `index.html` file in your browser

### Option 2: View the Main Paytm Clone Project
1. Navigate to the `Paytm-Homepage-Clone/` folder
2. Open `index.html` in your browser
3. The style will be automatically applied from `style.css`

**No build process or server required** – Simply open the HTML files directly in your browser.
