‚öîÔ∏è Scoping Breakdown

var
Functional scope. Allows redeclaration. The "Classic" way.
let
Block scope. No redeclaration. The "Modern" choice.
const
Block scope. Absolute fixed value. The "Secure" choice.

üå©Ô∏è The Temporal Dead Zone (TDZ)

Unlike `var`, which is hoisted with `undefined`, `let` and `const` are in a "Dead Zone" until their declaration is reached.

üë• Variable Shadowing

Shadowing occurs when a variable declared in an inner scope has the same name as a variable in an outer scope.
