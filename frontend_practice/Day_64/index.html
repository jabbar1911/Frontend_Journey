Day 64 – Git & GitHub

Topic: Version Control and Code Collaboration

What is Git?

Git is a distributed version control system used to track changes in source code and manage project history efficiently.

What is GitHub?

GitHub is a cloud-based platform that hosts Git repositories and enables collaboration between developers.

Git Setup & Configuration

git --version
git config --global user.name "Your Name"
git config --global user.email "your@email.com"

Creating a Local Repository

git init
git status

Staging and Committing Files

git add .
git commit -m "Initial commit"

Connecting to GitHub

git remote add origin https://github.com/username/repository-name.git
git branch -M main
git push -u origin main

Key Learnings

  • Git tracks code changes locally
  • Commits maintain project history
  • GitHub helps store and share code online
  • Essential for team collaboration
