# Day 73–75 - Capstone Project: Paytm Homepage Clone

## Objective
Create an accurate recreation of the Paytm homepage design using only HTML5 and CSS3, combining all previously learned skills into a single, real-world project.

## Files Included
- `index.html` – Main HTML structure
- `style.css` – Complete stylesheet
- `src/` – Folder containing all images and assets
  - Logo and branding images
  - Product images
  - Icons and illustrations
  - Background images

## Concepts Practiced

- Complete HTML semantic structure for a large project
- Complex CSS layouts using Flexbox and Grid
- Multi-section page design
- Navigation bar styling
- Hero sections with background images
- Product showcase sections
- Feature cards and components
- Footer design and styling
- Image asset management and organization
- Consistent color and typography across sections
- Responsive layout foundations
- Professional-grade code organization

## What I Learned

This capstone project was the culmination of everything I'd learned. Building a real-world website design from scratch required:

**Planning and Organization**
- Breaking down a complex design into manageable sections
- Organizing files and assets in a logical structure
- Planning the HTML structure before coding

**Technical Skills**
- Creating semantic, well-structured HTML that matches design requirements
- Writing extensive CSS to style multiple sections and components
- Managing many image assets and ensuring they display correctly
- Maintaining code readability and consistency throughout

**Design Thinking**
- Understanding how professional websites are structured
- Appreciating the attention to detail in real designs
- Learning that every pixel, color, and spacing choice has a purpose
- Recognizing patterns and components used across the design

**Problem-Solving**
- Figuring out how to recreate specific design elements with CSS
- Handling alignment and spacing challenges
- Making responsive decisions to adapt designs to different approaches

## Key Achievements

- Successfully recreated a recognizable, real-world website design
- Organized code in a professional folder structure
- Used semantic HTML throughout the project
- Created a comprehensive CSS stylesheet
- Properly managed and organized image assets
- Maintained consistency in design and styling
- Produced clean, readable code suitable for a portfolio

## Challenges and Solutions

**Challenge:** Replicating exact spacing and alignment
**Solution:** Used browser developer tools to inspect and compare, practiced with Flexbox and Grid

**Challenge:** Managing many image assets
**Solution:** Created a dedicated `src/` folder and organized images by type

**Challenge:** Keeping CSS organized for a large project
**Solution:** Used comments, grouped related styles, and maintained a logical structure
