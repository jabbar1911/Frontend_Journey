## What You'll Learn

By exploring these projects in order, you'll gain:

- **HTML Skills** - From basic structure to semantic markup and forms
- **CSS Fundamentals** - Selectors, properties, colors, fonts, and spacing
- **Media Integration** - Embedding and styling audio and video
- **Layout Techniques** - Flexbox, Grid, and responsive design foundations
- **Styling Best Practices** - Professional UI design and polish
- **Project Organization** - File structure and asset management
- **Progressive Enhancement** - Building skills that compound over time

## Tips for Learning

- **Read the individual Day README** - Each day folder has a dedicated README with learning objectives and insights
- **Inspect with Developer Tools** - Right-click and select "Inspect" to see how things are built
