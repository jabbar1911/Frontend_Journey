CSS Pseudo-Classes (Live Examples)

Pseudo-classes are used to style elements based on their state or user interaction. They help make web pages interactive without using JavaScript.

:hover

What: Applies styles when the mouse is placed over an element.
Why: Shows that an element is clickable.

CSS Pseudo-Elements (Live Examples)

Pseudo-elements allow styling specific parts of content without modifying HTML structure.

  • ::first-letter – styles first letter
  • ::selection – styles selected text
  • ::before / ::after – inserts decorative content

CSS Positioning (Live Examples)

CSS positioning controls where elements appear on the page. Different positioning types are used together in real projects.

  • position: sticky → Navbar stays on top while scrolling
  • position: relative → Content card positioning
  • position: absolute → “CSS” watermark inside card
  • z-index → Controls layer order

Form – :focus & ::placeholder
